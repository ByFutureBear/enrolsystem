# online-enrolment-system
PRG3204E Group Project


## How to Run This Project

Clone the repository
Download or clone this project from GitHub to your computer.

**git clone your-repo-link**

##
Install SQL Server Management Studio (SSMS)
Download and install SQL Server Management Studio 21.

Make sure your SQL Server service is running.

Import or create the project database if needed.
##
Open the project
Open the folder in Visual Studio Code (VS Code).

Run the project
In the VS Code terminal, type the following command:

**dotnet run**


Access the web app
After the command runs successfully, open your browser and go to:

**https://localhost:******


or whatever URL appears in the terminal.
